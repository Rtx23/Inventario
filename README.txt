Proyecto de Gestión de Inventarios

Descripción:
Este proyecto proporciona un sistema de gestión de inventarios para una tienda,
permitiendo la gestión de productos, proveedores, clientes y órdenes.

Estructura del Proyecto:
- producto.py: Definición de la clase Producto.
- proveedor.py: Definición de la clase Proveedor.
- cliente.py: Definición de la clase Cliente.
- orden.py: Definición de la clase Orden.
- utils.py: Funciones de utilidad para lectura y escritura en JSON.
- main.py: Menú principal e interacción con el usuario.
- data/: Directorio con los archivos JSON.
    - productos.json: Datos de productos.
    - proveedores.json: Datos de proveedores.
    - clientes.json: Datos de clientes.
    - ordenes.json: Datos de órdenes.
- README.txt: Este archivo con la descripción del proyecto.

Instrucciones de Uso:
1. Ejecutar `main.py` para interactuar con el sistema de gestión de inventarios.
2. Utilizar el menú para agregar, ver y gestionar productos, proveedores, clientes 
   y órdenes.
3. Los cambios se guardarán automáticamente en los archivos JSON en el directorio `data`.
